# Pre-Publish Checklist

Run before commit and push.

- [ ] Article contains explicit architecture tradeoff.
- [ ] At least 2 defended decisions are documented.
- [ ] No production-identifying details are present.
- [ ] Placeholder names are consistent (ZXX_*, SYSTEM_A/B, MIDDLEWARE, TX01/TX02).
- [ ] EN/DE/ES arguments are semantically aligned.
- [ ] Front matter fields are complete and valid.
- [ ] `lang` and `translations` metadata are consistent.
- [ ] Slugs are lowercase, ASCII-only, hyphenated.
- [ ] Description and social metadata are concrete and architecture-focused.
- [ ] Comments labels are localized correctly.
- [ ] Redirect files (if any) point to canonical routes.
