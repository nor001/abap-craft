# Post-Publish Checklist

Run after deployment to GitHub Pages.

- [ ] Canonical EN/DE/ES article routes resolve correctly.
- [ ] Legacy redirect URLs resolve to canonical routes.
- [ ] Language switch pills navigate correctly among all 3 versions.
- [ ] Giscus loads and maps by pathname.
- [ ] OG/Twitter metadata is present in page source.
- [ ] Homepage featured links point to current canonical posts.
- [ ] Posts listing page shows expected date and language tags.
- [ ] No broken internal links detected in new article set.
- [ ] No accidental exposure of sensitive production details.
